== Installation ==

Installing unblockadblocker is easy, go to you your WordPress admin panel and click on Plugins > Add New, searching for "unblockadblocker".
Alternatively, you can install the plugin manually by downloading the plugin from wordpress.org/plugins
1. Upload the entire unblockadblocker folder to the /wp-content/plugins/ directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
